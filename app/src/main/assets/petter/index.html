理赔程序

第一步：出险报案

发生保险事故后，被保险人应立即拨打平安24小时报案电话--95511提供事故相关情况，其中包括如下的内容：

1. 保单号

2. 意外发生时间、意外发生地点

3. 情况简述，包括意外如何发生、施救情况

4. 损失程度及估损金额

5. 现场联系人的姓名、联络方式

6. 平安在接到被保险人的损失通知后，将立即安排损失检验及定损工作。

第二步：查勘

平安在接到被保险人报案电话后， 理赔人员即与报案人联系并告知是否需要现场查勘，如需查勘会约定查勘时间，必要时会聘请公估人对案件进行处理。

第三步：提供索赔资料

被保险人根据现场理赔人/公估人的指引提供相关索赔资料，一般理赔需要的资料有：

1．保险单正本（及有关承保附件）复印件

2．出险通知书（被保险人签字）

3．赔款收据（填写帐号及签字，其他内容不填写）

4．现场（事故）照片

5．宠物到正规宠物医院就医的发票、病历、用药清单等

6．宠物丢失的报案回执

7．寻找宠物花费的广告费用发票、费用清单等。

理赔人/公估人会根据案件情况的不同指引被保险人准备相应索赔资料。

第四步：理赔人审核资料

理赔人审核被保险人提交的各种索赔资料，并视情况可能提出补充要求。

第五步：补充索赔资料，确保资料齐全

理赔人初步审核有关单证后，可进一步以书面形式提出补充证明要求，以便进一步确定损失情况。在初步审核单证后，即可出具初步理算报告，便于被保险人领取预付赔款。

第六步：出具理算报告

由理赔人出具正式理算报告，作为被保险人获取赔款的主要依据。

第七步：赔款

发生保险损失后，我们在收到理算人最终理算报告后，立即安排保险赔款．
