Add and delete device


1.Bind with new device

It will remind you to bind the device if this is the first time to use this device.

If you already bind successfully, please click user's alter ego on top left corner of homepage, swipe down to show device list, click" add new device" ,or come to "mine"click" devices management", click"+"icon in device management"page.

Add the device:

1.HT-790S、HT-770、MPIP-620、PT-718、PT-719、PT-690、213GD、213GL、213TH :click" scan QR code"in binding page.

2.PT-720: input SIM card number, QR code would show up once click "confirmation", it would bind successfully after scan QR code.

2.authorization

Come to "mine" page to click" account management" click "+"icon to add sub accounts (max. 3 sub accounts), note please sub accounts must be all registered to be added.

3.Unbind the device

Go to device management page to select the device which you want to delete. will come up when swipe left ,click it to unbind.
