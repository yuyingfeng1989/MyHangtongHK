Parameter Configuration

You can configure the device parameter to meet with different requirements.

(1)Location frequency：

ser can configure the data upload interval, the location frequency determine the device location data capture frequency, the higher frequency, the much more capture location data times. However, the higher frequency means the power consumption and the shorter the standby time. We suggest you to pay more attention on choosing the appropriate positioning frequency.

(2)APN Setting：

When the network is normal, the device was installed and the device is power on, if the Yellow LED and Green LED are normal blinking and you can't see any location data on the APP, it means you will need to set the APN.

APN(Access Point Name) is required to let Sim card surf the internet, it determine which access method for the Sim card to access the network. You can check with local Sim card operator for the APN.

APN configuration, Pls ensure the device Sim card no. is correct ( pls go to "Me—>Advanced setting—>APN setting—>device Sim no."to edit the sim card no.）. After input the APN, pls submit, then your mobile no. will receive setting results, if the message is "OK" means setting successfully, if the message is "failure" means setting failed.

(3)Alarm push settings：

User can set to receive different types of alarm push reminder.

(4)Time-zone setting：

If the GPS position time on the map is not your local time, user can go to Time-zone setting to set your own time-zone and adjust the time to be the right time.

(5)Reset：

If you click this Reset, device parameter configuration will restore factory reset except IP, Domain, Port, APN, APN Username, APN Password, Device ID.

(6)Check update：

User can check if new APP version and new device firmware are available to update. If your APP version is not the latest, you can do upgrade here.

(7)User feedback ：

Any feedback and suggestions of APP, hardware and services are welcome, we will try our best to improve our products and services.
